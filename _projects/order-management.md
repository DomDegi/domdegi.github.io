# Order and Inventory Management System

A procedural project built in C focusing heavily on low-level data structures and algorithmic efficiency.

## Implementation Details
- Applied Hash Tables for rapid inventory lookups.
- Utilized Heaps for priority-based order processing.
- Managed dynamic memory effectively using complex Linked Lists.
