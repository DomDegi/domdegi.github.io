# Grade Management Web Application

A full-stack architecture built to manage university exam cycles, tracking grades over semesters. 

## Key Features
- **Role-based Access:** Defined clear boundaries between Students and Professors.
- **Grade Entry:** Allowed professors to input, finalize, and publish grades dynamically.
- **Student Registration:** Comprehensive module for course registration and exams tracking.
